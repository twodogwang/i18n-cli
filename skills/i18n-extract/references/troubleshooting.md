# 常见问题

## 1. 提取后文件为空

**原因**：可能是配置文件路径不正确

**解决方案**：

- 检查 `input` 配置是否正确
- 使用绝对路径或相对于配置文件的路径

## 2. import 语句重复导入

**原因**：多次运行提取命令

**解决方案**：

- 使用 `incremental: true` 配置
- 或在提取前先回滚代码

## 3. Excel 导入失败

**原因**：Excel 格式不正确

**解决方案**：

- 确保 Excel 表头格式为 `['字典key', 'zh-CN', 'en-US']`
- 检查 `excelPath` 配置是否正确
